## Installation

```bash
npm i --force
```

## How to run

Development environment:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Production environment:

```bash
npm start
```

Test environment:

```bash
npm test
```

## Update packages

Install ncu:

```bash
npm install -g npm-check-updates
```

Verify dependencies:

```bash
ncu -u
```

Update them:

```bash
npm update
```
